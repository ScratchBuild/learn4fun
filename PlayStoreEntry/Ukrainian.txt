Title
=====

Learn 4 Fun - Математичні вправи



Short Description
=================

Виконай декілька математичних вправ перед використанням телефона чи планшета!


Description
===========

Learn 4 Fun працює як екран блокування. Перед розблокуванням телефона чи планшета, ваша дитина повинна спочатку виконати декілька математичних вправ та отримати за це час для ігор.

Як тільки час скінчиться, пристриій буде заблоковано і екран з вправами з'явиться знову.


Learn 4 Fun це не дитячий режим! Доступ до програм та інтернету не обмежується.


Вибір математичних операцій:
★ Додавання
★ Віднімання
★ Множення
★ Ділення
★ Перетворення величин
★ Числові шаблони


Додаткові можливості:
★ Статистика
★ Телефон завжди доступний (наприклад для екстрених викликів)
★ Захист паролем батьківських налаштувань
★ Невірно вирішені вправи будуть повторюватись частіше
★ Регульований діапазон чисел для додавання та віднімання
★ Вибір окремих серій для множення та ділення
★ Регульована кількість отримуваного часу для ігор
★ Регульована максимальна кількість часу для ігор

Перетворення величин:
★ Довжина
★ Вага
★ Об'єм (рідини)
★ Метрична чи Англійска системи мір

Google Play Games (необов'язково)
★ Розблокування досягнень
★ Список лідерів для кожної категорії

Дозволи:
Для коректної роботи, Learn 4 Fun необхідно виявляти чи працюють у фоні інші програми.
На пристроях Android 5.0+, ця функція вимагає доступу до використання даних.
